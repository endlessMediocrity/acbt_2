# Моделирование логических вентелей

Простое приложение для визуального создания и тестирования логических схем. Позволяет собирать схемы из логических элементов перетаскиванием блоков и сразу видеть результаты.

# Что умеет программа

- Добавлять логические блоки (AND, OR, XOR, NOT, NAND, NOR)
- Создавать входные (INPUT) и выходные (OUTPUT) элементы  
- Соединять блоки между собой
- Перетаскивать блоки по рабочей области
- Менять значения входов двойным кликом
- Автоматически вычислять результат схемы
- Показывать таблицу истинности для текущей схемы
